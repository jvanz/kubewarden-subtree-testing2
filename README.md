# Kubewarden
This is a monorepo for Kubewarden projects. And it has been generated by the following script:
```bash
#!/bin/bash
set -ex

pushd .

# clean previous runs
rm -rf kubewarden-monorepo

# create/fetch directories and code
mkdir kubewarden-monorepo

# init monorepo
echo "# Kubewarden" > kubewarden-monorepo/README.md
echo "This is a monorepo for Kubewarden projects. And it has been generated by the following script:" >> kubewarden-monorepo/README.md
{
  echo '```bash'
  cat -- "$(realpath -e -- "$0" || echo "$0")"
  echo '```'
} >> kubewarden-monorepo/README.md
cd kubewarden-monorepo
git init
git branch -m main
git add README.md
git commit -s -m "Initial commit"

# clone repositories to be merged
git subtree add -P policy-fetcher git@github.com:kubewarden/policy-fetcher.git main
git subtree add -P policy-evaluator git@github.com:kubewarden/policy-evaluator.git main
git subtree add -P policy-server git@github.com:kubewarden/policy-server.git main
git subtree add -P kwctl git@github.com:kubewarden/kwctl.git main
git subtree add -P kubewarden-controller git@github.com:kubewarden/kubewarden-controller.git main

# push monorepo to GitHub
git remote add origin git@github.com:jvanz/kubewarden-subtree-testing2.git
git push --force origin main

popd
```
